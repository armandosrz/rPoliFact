Topic Modeling

The topic modeling section will contain all the information gathered from the political fact checking subreddit. This model will create a number of topics from the text data gathered from each post. This text data includes the post title, self text (post body), comments, and text from linked articles. The text data is scraped, cleaned, normalized and given as input to the LDA model. The results give us a number of topics as a set. Each topic in the set includes words most frequently found together, in which we can generalize into a larger topic. This section will include two different types of analysis of topics. First, we will analyze general topics found from the data as a whole. This includes all posts within the subreddit. Second, we will use topic modeling for a given flair type (True, False, Mostly True...etc.)
